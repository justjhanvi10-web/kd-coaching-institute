# KD Coaching Institute

Official website for **KD Coaching Institute**, Road No. 25, Punjabi Bagh Club Road, New Delhi.

A static website (plain HTML/CSS/JS) for a coaching institute serving Classes 1–12 across Science, Commerce and Humanities.

## Pages
- **Home** — overview + 2027 admissions waitlist
- **Founder** — Priyanka Batra Gakhar
- **Faculty** — teaching faculty & support team
- **Fees & Timetable** — fee structure and class schedules, Nursery to Class 12
- **Achievers** — Students of the Year & toppers
- **Contact** — phone, email, address, map

## Local preview
Any static file server works, e.g.:

```bash
python -m http.server 5151
```

Then open http://localhost:5151
